**RootLayout changes:**
- Keep `i18nReady` useState and `initI18n()` useEffect (existing).
- Remove the `SplashScreen.hideAsync()` call from the `initI18n()` useEffect — splash hide is now delegated to `RootNavigator` which waits for BOTH i18n AND auth.
- Wrap existing providers with `<AuthProvider>` inside `<ThemeProvider>`.
- Replace `<Slot />` with `<RootNavigator />`.

**RootNavigator (new internal component, defined in same file):**
```typescript
function RootNavigator() {
  const { session, isLoading } = useAuth();

  useEffect(() => {
    if (!isLoading) {
      SplashScreen.hideAsync().catch(() => {});
    }
  }, [isLoading]);

  if (isLoading) return null;

  return (
    <Stack screenOptions={{ headerShown: false }}>
      <Stack.Protected guard={!session}>
        <Stack.Screen name="(auth)" />
      </Stack.Protected>
      <Stack.Protected guard={!!session}>
        <Stack.Screen name="(tabs)" />
      </Stack.Protected>
    </Stack>
  );
}
```

- Import `Stack` from `expo-router` (replaces `Slot` import).
- Import `AuthProvider` and `useAuth` from `@/contexts/AuthContext`.
- `RootNavigator` returns `null` while `isLoading` — splash screen stays visible during this time.
- Once `isLoading` is false, `SplashScreen.hideAsync()` fires and the guard evaluates:
  - `session` is null → shows `(auth)` group (sign-in screen)
  - `session` is truthy → shows `(tabs)` group (tab bar)

2. Delete or replace `BalanceTracker/app/index.tsx`:
   - This file was the Phase 7 test screen. It must be removed because Expo Router's file-system routing will now resolve `/` to `(tabs)/index.tsx` when authenticated. If `app/index.tsx` exists alongside `app/(tabs)/index.tsx`, it will conflict.
   - Delete the file entirely.

**Critical anti-patterns to avoid:**
- Do NOT use `useRouter().replace()` for auth redirect — Stack.Protected handles this.
- Do NOT call `queryClient.clear()` in `onAuthStateChange` — only in explicit `signOut()` (already handled in Plan 01).
- Do NOT show `<Slot />` or any content while `isLoading` — the splash screen covers this gap.
