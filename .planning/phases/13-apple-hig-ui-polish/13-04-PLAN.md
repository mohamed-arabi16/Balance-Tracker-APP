}

import { SymbolView } from 'expo-symbols';

```tsx
{symbolName ? (
  <SymbolView
    name={symbolName as any}
    tintColor="#9ca3af"

    style={{ marginBottom: 20 }}
  />
) : null}
```

**4. Make CTA conditional:**
The current CTA button is always rendered. Wrap it:
```tsx
{ctaLabel && onCta ? (
  <TouchableOpacity
    onPress={handlePress}
    style={styles.button}
    className="bg-blue-600 dark:bg-blue-500 rounded-xl py-3"
    accessibilityRole="button"
  >
    <Text className="text-white text-base font-semibold text-center">
      {ctaLabel}
    </Text>
  </TouchableOpacity>
) : null}
```

**5. Update handlePress — guard against undefined onCta:**
```typescript
function handlePress() {
  haptics.onToggle();
  onCta?.();
}
```

**6. Verify safe SF Symbol names for finance app:**
The following names are confirmed available on iOS 15+:
- Income/transactions: `'arrow.up.circle'`
- Expenses: `'arrow.down.circle'`
- Debts: `'creditcard'`
- Assets: `'banknote'`
- Dashboard: `'chart.bar'`

Use `as any` type assertion on the `name` prop since the type is a very long union — using `as any` avoids TS errors without weakening safety in meaningful ways (expo-symbols silently shows nothing for invalid names).
