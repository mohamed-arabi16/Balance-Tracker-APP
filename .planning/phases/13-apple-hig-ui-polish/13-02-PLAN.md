**Both files share the same set of changes — apply identically:**

**1. Import additions:**
- Add `{ useColorScheme }` from `'nativewind'`
- Add `{ COLORS }` from `'@/lib/tokens'`
- Add `{ Stack }` from `'expo-router'` (for headerRight)
- Add `{ TouchableOpacity }` to react-native imports if not present

**2. SafeScreen — switch to grouped bg:**
- On the main screen's SafeScreen, add `grouped` prop:
  - `<SafeScreen edges={['bottom']} grouped>` (Transactions already has edges bottom, keep it)
  - For ExpenseScreen (which is a component inside index.tsx, not a full screen): the SafeScreen wrapping is in index.tsx — ExpenseScreen only renders the FlatList + FAB. Remove the `<SafeScreen edges={['bottom']}>` from ExpenseScreen — instead, TransactionsScreen's SafeScreen wraps both tabs.

**3. Row dark mode via useColorScheme:**
In the component body (IncomeRow and ExpenseRow respectively), add:
```typescript
const { colorScheme } = useColorScheme();
const rowBg = colorScheme === 'dark' ? COLORS.cellBg.dark : COLORS.cellBg.light;
```
Then pass `rowBg` into the row style:
```typescript
style={({ pressed }) => [
  styles.row,
  { backgroundColor: rowBg },
  { opacity: pressed ? 0.7 : 1 },
]}
```
Remove the hardcoded `backgroundColor: '#ffffff'` from the `row` StyleSheet entry (set to `undefined` or remove the key).

**4. Row minimum height:**
In the `row` StyleSheet entry, add `minHeight: 44`.

**5. Row press feedback:**
Change `<Pressable onPress={handlePress} style={styles.row}>` to use a style function:
```typescript
<Pressable
  onPress={handlePress}
  style={({ pressed }) => [
    styles.row,
    { backgroundColor: rowBg },
    { opacity: pressed ? 0.7 : 1 },
  ]}
>
```

**6. Separator styling:**
Change separator from `height: 1, backgroundColor: '#f3f4f6'` to:
```typescript
separator: {
  height: StyleSheet.hairlineWidth,
  backgroundColor: '#C6C6C8',
  marginStart: 16,
},
```

**7. Status badge hitSlop for 44pt:**
Add `hitSlop={{ top: 10, bottom: 10, left: 10, right: 10 }}` to the StatusBadge Pressable in both files.

**8. Remove FAB — add headerRight nav button instead:**
- Remove the `<TouchableOpacity ... style={styles.fab}>` and `<Text style={styles.fabText}>` JSX from both IncomeScreen and ExpenseScreen
- Remove `fab` and `fabText` from StyleSheet
- In TransactionsScreen (index.tsx), add Stack.Screen headerRight:
  ```tsx
  import { Stack, useRouter } from 'expo-router';
  // Inside TransactionsScreen component:
  <Stack.Screen
    options={{
      headerRight: () => (
        <TouchableOpacity
          onPress={() => router.push(
            activeTab === 'income'
              ? '/(tabs)/transactions/add-income'
              : '/(tabs)/transactions/add-expense' as any
          )}
          style={{ paddingHorizontal: 8 }}
          accessibilityRole="button"
          accessibilityLabel={activeTab === 'income' ? 'Add Income' : 'Add Expense'}
        >
          <Text style={{ color: '#007AFF', fontSize: 17 }}>+</Text>
        </TouchableOpacity>
      ),
    }}
  />
  ```

**9. Tab bar dark mode:**
In TransactionsScreen styles, fix tabBar background:
```typescript
tabBar: {
  flexDirection: 'row',
  paddingHorizontal: 16,
  paddingVertical: 12,
  gap: 8,
  borderBottomWidth: StyleSheet.hairlineWidth,
  borderBottomColor: '#C6C6C8',
  // Remove hardcoded backgroundColor: '#ffffff' — let parent SafeScreen bg show through
},
```
