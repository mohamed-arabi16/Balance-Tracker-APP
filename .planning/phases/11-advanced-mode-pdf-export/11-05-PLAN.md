**ClientPickerField component (inline in each form or extracted):**

Use React Native's `@react-native-picker/picker` if installed. Otherwise build a simple custom picker:
- A `TouchableOpacity` that shows the selected client name (or "Select Client (optional)")
- Opens a `Modal` with a `FlatList` of clients
- "No Client" option at top of list to clear selection
- Close modal on selection

```tsx
{isAdvanced && (
  <View style={{ marginBottom: 16 }}>
    <Text>{t('transactions.form.client', 'Client (Optional)')}</Text>
    <TouchableOpacity
      onPress={() => setClientModalVisible(true)}
      style={{ borderWidth: 1, borderColor: '#ccc', padding: 12, borderRadius: 8 }}
    >
      <Text>
        {selectedClientId
          ? clients.find(c => c.id === selectedClientId)?.name ?? 'Select Client'
          : t('transactions.form.noClient', 'No Client')}
      </Text>
    </TouchableOpacity>
    <Modal visible={clientModalVisible} transparent animationType="slide">
      <View style={{ flex: 1, backgroundColor: 'rgba(0,0,0,0.5)', justifyContent: 'flex-end' }}>
        <View style={{ backgroundColor: 'white', maxHeight: '60%', borderTopLeftRadius: 16, borderTopRightRadius: 16 }}>
          <FlatList
            data={[{ id: null, name: t('transactions.form.noClient', 'No Client') }, ...clients]}
            keyExtractor={item => item.id ?? 'none'}
            renderItem={({ item }) => (
              <TouchableOpacity
                onPress={() => { setSelectedClientId(item.id); setClientModalVisible(false); }}
                style={{ padding: 16, borderBottomWidth: 1, borderBottomColor: '#eee' }}
              >
                <Text>{item.name}</Text>
              </TouchableOpacity>
            )}
          />
        </View>
      </View>
    </Modal>
  </View>
)}
```
