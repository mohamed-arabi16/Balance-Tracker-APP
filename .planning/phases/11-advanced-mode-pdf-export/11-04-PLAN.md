**pdfTemplate.ts (src/lib/pdfTemplate.ts):**

Create a pure function `generateInvoiceHtml(invoice: InvoiceWithItems, clientName: string): string` that returns a complete HTML document string suitable for `expo-print`.

CRITICAL constraints (from RESEARCH.md):
1. NO local file:// URLs — all styles inline, system fonts only (-apple-system, Arial)
2. English-only output — per i18n design decision (resources.ts has `invoices.pdf.arabicNotice` key signaling PDF is always EN)
3. Must compute subtotal, tax, and total client-side from `invoice.items` — do NOT read `invoice.total` (it can be null as a generated column)
4. Use `Number(item.amount ?? item.quantity * item.unit_price)` for each line item amount

**PDF Export (EXPRT-02):**
```typescript
import * as Print from 'expo-print';
import * as Sharing from 'expo-sharing';

async function handleExportPdf() {
  try {
    const clientName = clientMap[invoice.client_id] ?? 'Unknown Client';
    const html = generateInvoiceHtml(invoice, clientName);
    const { uri } = await Print.printToFileAsync({ html });
    await Sharing.shareAsync(uri, {
      UTI: 'com.adobe.pdf',
      mimeType: 'application/pdf',
      dialogTitle: `Invoice ${invoice.invoice_number}`,
    });
  } catch (error) {
    haptics.onError();
    Alert.alert('Export Failed', (error as Error).message);
  }
}
```
