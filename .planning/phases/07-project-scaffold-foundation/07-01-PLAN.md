    2. cd into `BalanceTracker/` and install all dependencies:

    Production deps:
    ```
    npx expo install @supabase/supabase-js react-native-url-polyfill expo-sqlite @react-native-async-storage/async-storage @tanstack/react-query expo-router react-native-safe-area-context react-native-screens expo-linking expo-constants expo-status-bar nativewind expo-localization i18next react-i18next expo-haptics expo-blur expo-splash-screen react-native-keyboard-controller react-native-reanimated react-native-gesture-handler
    ```

    Dev deps:
    ```
    npm install -D tailwindcss@3.3.2 @react-native-community/eslint-config eslint
    ```

    3. Configure `app.json`:
    - Set `"orientation": "portrait"` (UX-07)
    - Set `"userInterfaceStyle": "automatic"` (THEME-01)
    - Set `"scheme"` for Expo Router deep linking
    - Set `"plugins"` array with `"expo-router"`, `"expo-localization"`
    - Set `"jsEngine": "hermes"`

    4. Configure `babel.config.js`:
    - Add `'nativewind/babel'` to plugins array
    - Add `'expo-router/babel'` to plugins if required by Expo Router v3

    5. Configure `metro.config.js`:
    - Import `withNativeWind` from `nativewind/metro`
    - Wrap the default Expo Metro config: `module.exports = withNativeWind(config, { input: './global.css' })`

    6. Create `tailwind.config.js`:
    - Content paths: `["./src/**/*.{js,jsx,ts,tsx}", "./app/**/*.{js,jsx,ts,tsx}"]`
    - Preset: `require('nativewind/preset')`
    - Theme extend as needed for custom colors

    7. Create `global.css` with Tailwind directives:
    ```css
    @tailwind base;
    @tailwind components;
    @tailwind utilities;
    ```

    8. Configure `tsconfig.json`:
    - Add path alias: `"@/*": ["./src/*"]`
    - Ensure `"strict": true`
    - Extend from `expo/tsconfig.base`

    9. Create `.eslintrc.js`:
    - Extend `@react-native-community`
    - Enable rule `"react-native/no-raw-text": "error"` (P8 pitfall — catches bare strings outside Text components)

    10. Create a minimal `app/_layout.tsx` that imports `global.css` and renders a `<Slot />` to verify the app compiles. Import `react-native-url-polyfill/auto` as the FIRST import (P1 pitfall — URL polyfill must be first).

    11. Create a minimal `app/index.tsx` that renders a `<View>` with a `<Text>` using a NativeWind class (e.g., `className="text-lg font-bold"`) to verify Tailwind rendering works.
