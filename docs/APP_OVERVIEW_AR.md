# نظرة عامة على تطبيق "تتبع الرصيد" (Balance Tracker)

تطبيق "تتبع الرصيد" هو تطبيق شامل لإدارة الأمور المالية الشخصية، مصمم لمساعدة المستخدمين على تتبع أصولهم وديونهم ودخلهم ومصروفاتهم في مكان واحد. يوفر التطبيق صورة واضحة للحالة المالية من خلال لوحة تحكم بديهية وصفحات تتبع مفصلة.

## المميزات الأساسية

- **لوحة التحكم**: نظرة شاملة على حالتك المالية، بما في ذلك إجمالي صافي الثروة، وتوزيع الأصول، وأحدث المعاملات.
- **إدارة الأصول**: تتبع أنواعاً مختلفة من الأصول، بما في ذلك النقد، والحسابات البنكية، والاستثمارات، والأصول المادية مثل الذهب أو العقارات.
- **تتبع الديون**: متابعة الالتزامات والقروض وأرصدة بطاقات الائتمان.
- **تتبع الدخل والمصروفات**: تصنيف ومراقبة التدفق النقدي لفهم عادات الإنفاق وأنماط الربح.
- **دعم لغات متعددة**: مترجم بالكامل للغتين الإنجليزية والعربية مع دعم الاتجاه من اليمين إلى اليسار (RTL).
- **تطبيق ويب تقدمي (PWA)**: قابل للتثبيت على الأجهزة المحمولة وأجهزة الكمبيوتر للوصول بدون إنترنت وتجربة تشبه التطبيقات الأصلية.

## التقنيات المستخدمة

- **الواجهة الأمامية**: React مع Vite لتجربة تطوير سريعة وحديثة.
- **التنسيق**: Tailwind CSS للتصميم المتجاوب و Shadcn/UI لمكونات متميزة وسهلة الاستخدام.
- **قاعدة البيانات والخدمات الخلفية**: Supabase (PostgreSQL) لتخزين البيانات والمصادقة في الوقت الفعلي.
- **إدارة الحالة**: React Query (TanStack Query) لجلب البيانات وتخزينها مؤقتاً بكفاءة.
- **الترجمة**: i18next لإدارة الترجمات.

## بنية التطبيق

يتبع التطبيق بنية حديثة تركز على الواجهة الأمامية:

1. **الصفحات**: تقع في `src/pages/` وتمثل المسارات المختلفة مثل لوحة التحكم، الأصول، إلخ.
2. **المكونات**: عناصر واجهة مستخدم قابلة لإعادة الاستخدام تم بناؤها باستخدام Shadcn/UI و Tailwind CSS في `src/components/`.
3. **الخطافات (Hooks)**: خطافات React مخصصة في `src/hooks/` للمنطق المشترك وجلب البيانات.
4. **السياقات (Contexts)**: إدارة الحالة العالمية للمظاهر والمصادقة وإعدادات التخطيط في `src/contexts/`.
5. **قاعدة البيانات**: يتم إدارتها بواسطة Supabase مع أمن مستوى الصف (RLS) لحماية البيانات.

## بدء العمل

1. تثبيت التبعيات: `npm install`
2. إعداد متغيرات البيئة في ملف `.env` (راجع `.env.example`).
3. تشغيل خادم التطوير: `npm run dev`
